# Online-Shop
Mini E-Commerce Website

A modern and responsive online shopping website built using HTML, CSS, JavaScript, and Bootstrap.  
This project includes multiple pages such as Home, Shop, Cart, Checkout, About, and Contact with interactive shopping features.

---

## Features

- Responsive design for all devices
- Dark mode toggle
- Product categories
- Add to cart functionality
- Shopping cart page
- Checkout page
- Contact page
- About page
- Modern UI with animations
- Bootstrap 5 integration
- Font Awesome icons
